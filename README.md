# My Library

## How to add a new book

1. Drop the PDF file into the `books/` folder.
2. Open `books.json` and add a new entry:

```json
{
  "title": "Book Title Here",
  "file": "books/yourfile.pdf",
  "tag": "Optional category"
}
```

3. Commit and push. The site updates automatically within a minute or two.

## Delete the sample entry

`books.json` currently has one placeholder entry ("Example Book — Delete Me").
Remove it once you've added your own books.
